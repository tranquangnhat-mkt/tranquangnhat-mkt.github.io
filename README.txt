TRẦN QUANG NHẬT — PORTFOLIO DEMO

Cách xem:
1. Giải nén file ZIP.
2. Mở file index.html bằng Chrome, Edge, Firefox hoặc Safari.
3. Website hoạt động trực tiếp, không cần cài đặt thêm.

Các tính năng trong bản demo:
- Giao diện responsive cho máy tính và điện thoại
- Chế độ sáng/tối
- Menu điều hướng mượt
- Bộ lọc dự án
- Cửa sổ case study
- Hiệu ứng xuất hiện khi cuộn
- Form liên hệ minh họa

Lưu ý:
- Nội dung và số liệu hiện tại là nội dung mẫu.
- Form liên hệ và nút tải CV chưa kết nối hệ thống thật.
- Toàn bộ website nằm trong một file index.html để dễ chỉnh sửa bằng vibe coding.
